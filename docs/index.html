AI Copilot

Natural language to OpenSCAD. Works with Claude and GPT. Bring your own API keys - they stay encrypted in your OS keychain.

Live 3D Preview

Interactive mesh viewer with orbit controls. Instant re-renders with content-hash caching. Real-time error diagnostics.

Smart Formatting

Tree-sitter based formatter for idempotent, AST-aware code formatting. Auto-format on save, preserves comments, 137+ tests.

First-Class 2D Support

Create 2D designs with SVG preview. Export to DXF for laser cutting or CNC. Automatic 2D/3D mode detection.
